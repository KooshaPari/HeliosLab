# Support

## Getting Help

For questions, bug reports, and feature requests, please use the project's
GitHub issue tracker:

- **Issues:** https://github.com/KooshaPari/HeliosLab/issues

Before opening a new issue, search existing issues to avoid duplicates.

## Security Issues

**Do not** report security vulnerabilities through public issues. See
[`SECURITY.md`](./SECURITY.md) for the private reporting process.

## Community

- **Discussions:** https://github.com/KooshaPari/HeliosLab/discussions

## Commercial Support

This project is community-supported. No commercial support contracts are
offered by the maintainers.
